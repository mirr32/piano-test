# 楽譜ページの使用音確認（目視）

このファイルは、ユーザー提供画像（ソナタ悲愴 第2楽章、該当ページ）の目視確認に基づく簡易検証メモです。

## 1. 調号の確認
- 調号は **変イ長調（A♭ major）**で、**♭×4（B♭/E♭/A♭/D♭）**が付いているのを確認。  

## 2. 画像から確認できた音
### 2.1 調号内の音（ダイアトニック）
- **C / D♭ / E♭ / F / G / A♭ / B♭** は譜面全体に頻出。  

### 2.2 臨時記号で確認できた音
- **E♮（Eナチュラル）**: ナチュラル記号付きのEが複数箇所に見える。  
- **G♭（Gフラット）**: 低声部にGへの♭が付いた形が確認できる。  
- **G♮（Gナチュラル）**: G♭の後に打ち消しとしてナチュラルが付いた箇所が見える。  

> ※ 画像解像度と角度の都合で、すべての小節を完全に断定できるわけではありませんが、上記の臨時記号は目視で確認できました。

## 3. 画像内で見当たらなかった音（目視範囲）
- **C♭ / F♭ / B♮** は明確には見当たりませんでした。  

## 4. まとめ
- ご提示の「使用音まとめ（9音）」は、**目視確認の範囲では妥当**です。  
- ただし、**最終確定のためには**高解像度スキャンや拡大画像での再確認を推奨します。
